# GNNs_learning

A repository for learning and experimenting with **Graph Neural Networks (GNNs)**.

## What We're Learning

- Fundamentals of Graph Neural Networks
- Message passing and graph convolution operations
- Common GNN architectures: GCN, GNNs, Graph Attention and more
- Applications of GNNs on node classification, link prediction, and graph classification tasks
- Hands-on implementation using PyTorch and PyTorch Geometric

1. [UvA Deep Learning Course – Tutorial Notebooks](https://uvadlc-notebooks.readthedocs.io/en/latest/tutorial_notebooks) – Practical tutorial notebooks covering deep learning concepts including GNNs, from the University of Amsterdam.
